Listas Ordenadas

Lista ordenada: Uma lista ordenada é apenas uma lista de coisas organizadas em uma ordem específica, Cada item na lista recebe um número ou marcador especial para mostrar sua posição.

Listas não ordenadas

Lista não Ordenadas: A lista não ordenada os itens não têm uma ordem definida e são marcados com um símbolo especial, como um ponto (•).


Tipos de textos utilizado em HTML

3 TAGS de texto utilizadas por progamadores.

Título: Para título, utilizamos as TAGS: <h1> , <h2> , <h3> , <h4> , <h5> , <h6> que serve para deixar o texto maior, tem diversas formas de utilizar e modificar essas TAG, em CSS fica a estetica do Site, o HTML seria somente a base/estrutura do conteudo.

Paragrafos: Para paragrafos, nós progamadores utilizamos a TAG:<P> entre as tags podemos inserir um texto ou algo do tipo. tambem há formas de modificar ela no CSS, pode deixar le centralizada, cor, tamanho, fonte e ETC.


Lista ordenada: Para saber o que é uma lista Ordenada (Clique aqui)
Exemplos de como funciona a TAG:
<OL>

<li> Conteudo.....<li>
<li> Conteudo.....<li>
<li> Conteudo.....<li>
</OL>
↓ Ele saira desta forma ↓
  1. Conteudo.....
  2. Conteudo.....
  3. Conteudo.....

Lista não Ordenadas:

Para saber o que não é uma lista Ordenada (Clique aqui)
Exemplos de como funciona a TAG:
<ul>
<li> Conteudo.....<li>
<li> Conteudo.....<li>
<li> Conteudo.....<li>
</ul>
↓ Ele saira desta forma ↓
  • Conteudo.....
  • Conteudo.....
  • Conteudo.....

É importante ressaltar que o conteudo que for inserir no Site precisa estar dentro das tags!!!
